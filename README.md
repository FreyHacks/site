## 🌟 Technologies

- NextJS
- React
- Tailwindcss
- Preact
- React-Scroll
- React-Countdown-Hook

## ➡️ Development

1. Install Dependencies

   ```sh
   yarn
   ```

2. Start NextJs Development Server

   ```
   yarn dev
   ```
